FAQ

Q: Do I need all 5 API keys?
A: No. One key (DeepSeek) is enough to start. The other 4 act as your safety net.


Q: Will this slow down Claude Code?
A: No measurable difference. <1ms routing overhead, runs on localhost.


Q: Is this safe?
A: Yes. API keys stay on your machine. The router only proxies requests to providers you configure.


Q: What about updates?
A: npm update -g deepseek-router — your config stays untouched.


Q: Can I add custom providers?
A: Yes. Edit ~/.deepseek-router/router.config.json and add your own endpoints.


Q: VS Code extension?
A: Yes. Set ANTHROPIC_BASE_URL and ANTHROPIC_API_KEY in your shell before launching VS Code.
