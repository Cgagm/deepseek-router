常见问题

摸鱼和 ChatGPT 有什么不同？

摸鱼内置了 9 个国内 AI 模型，会根据你的任务自动选最优的一个。比如写代码用 DeepSeek，翻译用千问，写长文用 Kimi。ChatGPT 只有一个模型。而且摸鱼的数据存在本地，更安全。

买断制和 Token 版有什么区别？

功能完全一样。买断制 ¥69 永久使用，但需要你自己去各平台注册 API Key；Token 版不需要填 Key，买了就能用，余额用完再充值。动手能力强选买断，不想折腾选 Token。

需要什么样的电脑？

Windows 10/11，macOS 12+。4GB 内存即可。安装包约 150MB。

支持 Mac 吗？

首发 Windows。Mac 版正在开发中，预计 2 周内发布。

我的数据安全吗？

API Key 和聊天记录都存在你的电脑上，不会上传到我们的服务器。请求直接发到 AI 供应商，摸鱼不经过你的数据。买断版更是完全离线可用（本地填 Key）。

可以退款吗？

买断版 7 天内无理由退款。Token 版余额未使用可退。
